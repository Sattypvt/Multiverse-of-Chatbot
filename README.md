# 🤖 MULTIVERSE OF CHATBOT


An interactive AI chatbot built with **Python**, **Streamlit**, and **Google Gemini API** as part of the **MirAI School of Technology – Virtual Summer Internship 2026 (AI Builder Track)**.

The application allows users to choose different AI personalities and chat with Gemini in each selected persona.

---

## 🚀 Features

- 🤖 Multiple AI Personas
  - Teacher
  - Coding Expert
  - Motivational Coach
  - Comedian
  - Pirate
  - Shakespeare

- 💬 Interactive Chat Interface
- 📝 Chat History
- 🧹 Clear Chat Button
- ⏳ AI Loading Spinner
- ⚠️ Input Validation
- 📊 Response Statistics
- 🎨 Clean Streamlit UI
- 🔐 Secure Gemini API Integration

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Persona-Chatbot.git
```

### 2. Go to the project folder

```bash
cd AI-Persona-Chatbot
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add your Gemini API Key

Either:

Create a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

or directly paste your API key inside `app.py`.

---

### 5. Run the application

```bash
streamlit run app.py
```
