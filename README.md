# 🤖 Multiverse of Chatbot

A stateful AI chatbot built using **Streamlit** and **Google Gemini API** as part of the **MirAI School of Technology Virtual Summer Internship 2026**.

---

## 🚀 Features

- 🤖 Multiple AI Personas
  - Teacher
  - Coding Expert
  - Motivational Coach
  - Comedian
  - Pirate
  - Shakespeare

- 💬 Stateful conversation using `st.session_state`
- 🧠 Remembers previous messages
- 🎨 Beautiful Streamlit UI
- 📥 Download AI responses
- 💾 Download chat history
- 📊 Response statistics
- 🗑️ Clear chat option
- 🔒 Secure API key management using `.env`

---

## 🛠️ Technologies Used

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/Sattypvt/Multiverse-of-Chatbot.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

4. Run the application

```bash
streamlit run app.py
```
